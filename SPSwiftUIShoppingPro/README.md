# SPSwiftUIShoppingPro

一个使用 SwiftUI 开发的 iOS 购物应用项目。

## 项目简介

SPSwiftUIShoppingPro 是一个基于 SwiftUI 框架构建的现代化购物应用，旨在提供流畅的用户体验和优雅的界面设计。

## 技术栈

- **开发语言**: Swift
- **UI 框架**: SwiftUI
- **最低支持版本**: iOS (根据项目配置)
- **开发工具**: Xcode 16.4
- **数据库**: Realm (RealmSwift)

## 依赖项

项目使用以下 Swift Package Manager 依赖：

- **Realm**: 用于本地数据存储和管理
- **RealmSwift**: Realm 的 Swift 封装

## 项目结构

```
SPSwiftUIShoppingPro/
├── SPSwiftUIShoppingPro/
│   ├── Assets.xcassets/          # 资源文件
│   ├── ContentView.swift         # 主视图
│   └── SPSwiftUIShoppingProApp.swift  # 应用入口
└── SPSwiftUIShoppingPro.xcodeproj/   # Xcode 项目文件
```

## 环境要求

- macOS (用于开发)
- Xcode 16.4 或更高版本
- iOS 设备或模拟器

## 安装与运行

1. 克隆项目到本地：
```bash
git clone <repository-url>
cd SPSwiftUIShoppingPro
```

2. 使用 Xcode 打开项目：
```bash
open SPSwiftUIShoppingPro.xcodeproj
```

3. 等待 Swift Package Manager 自动下载依赖项

4. 选择目标设备或模拟器

5. 点击运行按钮 (⌘R) 或使用快捷键运行项目
